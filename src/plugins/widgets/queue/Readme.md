# Queue widget.

The Queue widget displays the number of messages on a message queue. 

Use the *widgetType* 'queue' to use the Queue widget.

```
{
  plugin: 'generic',
  widgetType: 'queue',
  datasourceId: 'MyDatasource',
  displayName: 'Something',
  fieldName: 'foo'
}
```

See also [widgets](../).
